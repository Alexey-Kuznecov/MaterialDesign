# MySetGulp
Configure packages such as npm, bower, sass, gulp, and more.

## Install
1. git clone https://github.com/AlexsofarBOT/MySetGulp.git
2. npm i
3. bower i

## Install plugins:
* gulp
* gulp-sass
* gulp-concat
* gulp-uglifyjs
* gulp-cssnano
* gulp-rename
* gulp-imagemin
* gulp-cach
* gulp-autoprefixer
* browser-sync
* imagemin-pngquant
* del
